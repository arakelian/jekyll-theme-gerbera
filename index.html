Gerbera

        

      

    

    

      
A documentation theme for Jekyll

      

        Gerbera is a Jekyll theme for creating documentation. Gerbera borrows heavily 
        from Facebook's GraphQL documentation style sheet, and allows you to create 
        modern, beautiful documentation quickly.
